âœ¨ Features

Everything you need to efficiently manage your global npm packages across different package managers.

List Global Packages

View all globally installed packages from any package manager with clean, formatted output.

Update Individual Packages

Update specific packages to their latest versions with clear progress feedback and error handling.

Bulk Update All

Update all global packages at once with automatic conflict resolution and detailed progress tracking.

Check Latest Version

View the latest version of any package in the npm registry without installing it.

Multi-Package Manager

Works seamlessly with npm, pnpm, Yarn, and Bun. Use the --pm flag to specify your preferred package manager.

Type Safe

Built with TypeScript for strict type checking and enhanced developer experience.
