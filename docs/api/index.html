API Reference

Complete TypeScript API documentation for npm-updater, including interfaces, types, and programmatic usage examples.

üìã Overview

npm-updater is built with TypeScript and provides a clean, type-safe API for programmatic usage. The API is organized around package manager support, command execution, and configuration management.

Main Entry Point

import { run } from "npm-updater";

await run();

Package Manager Utilities

import {
  validatePackageManager,
  getPackageManager,
  getPackageManagerConfig,
  formatPackageManagerList,
} from "npm-updater";

Command Functions

import { runls } from "npm-updater/commands/ls";
import { runupdate } from "npm-updater/commands/update";
import { runupdateall } from "npm-updater/commands/updateall";
import { showlatestversion } from "npm-updater/commands/latestversion";
import { showversion } from "npm-updater/commands/version";
import { showabout } from "npm-updater/commands/about";

üì¶ PackageManager Type

Union type representing all supported package managers. 

export type PackageManager = "npm" | "pnpm" | "yarn" | "bun";

Supported Package Managers

  • npm - Node.js default package manager
  • pnpm - Fast, disk space efficient package manager
  • yarn - Fast, reliable, and secure dependency management
  • bun - Ultra fast JavaScript runtime and package manager

Usage

import type { PackageManager } from "npm-updater";

const packageManager: PackageManager = "pnpm";
const supportedManagers: PackageManager[] = ["npm", "pnpm", "yarn", "bun"];

‚öôÔ∏è PackageManagerConfig Interface

Configuration interface for package manager-specific commands and behaviors. 

export interface PackageManagerConfig {
  name: PackageManager;
  displayName: string;
  listCommand: string;
  listJsonCommand: string;
  installCommand: (packageName: string, version?: string) => string;
  viewCommand: (packageName: string, version?: string) => string;
}

Properties

name

PackageManager

The package manager identifier.

displayName

string

Human-readable name for the package manager.

listCommand

string

Command to list globally installed packages in human-readable format.

listJsonCommand

string

Command to list globally installed packages in JSON format.

installCommand

(packageName: string, version?: string) => string

Function that generates the install/update command for a package.

viewCommand

(packageName: string, version?: string) => string

Function that generates the view command to check package information.

Example

import { getPackageManagerConfig, type PackageManager } from "npm-updater";

const config = getPackageManagerConfig("npm");
console.log(config.installCommand("prettier"));
// Output: "npm install -g prettier@latest"

console.log(config.viewCommand("react", "18.0.0"));
// Output: "npm view react@18.0.0 version"

üóÇÔ∏è PACKAGE_MANAGERS Constant

Object containing configurations for all supported package managers. 

export const PACKAGE_MANAGERS: Record

Access Package Manager Configurations 

import { PACKAGE_MANAGERS } from "npm-updater";

// Access npm configuration
const npmConfig = PACKAGE_MANAGERS.npm;

// Access pnpm configuration
const pnpmConfig = PACKAGE_MANAGERS.pnpm;

// List all supported managers
const allConfigs = Object.values(PACKAGE_MANAGERS);

Available Configurations

npm

listCommand: "npm list -g"
listJsonCommand: "npm ls -g --json"
installCommand: "npm install -g @"

pnpm

listCommand: "pnpm list -g"
listJsonCommand: "pnpm list -g --json"
installCommand: "pnpm add -g @"

yarn

listCommand: "yarn global list"
listJsonCommand: "yarn global list --json"
installCommand: "yarn global add @"

bun

listCommand: "bun list -g"
listJsonCommand: "bun list -g --json"
installCommand: "bun add -g @"

üîß Utility Functions

validatePackageManager

export function validatePackageManager(pm: string): PackageManager;

Validates that the provided package manager is supported and returns the validated type.

Parameters

  • pm - Package manager name to validate

Returns

Validated PackageManager type

Example

import { validatePackageManager } from "npm-updater";

try {
  const manager = validatePackageManager("pnpm");
  console.log(`Validated package manager: ${manager}`);
} catch (error) {
  console.error("Invalid package manager:", error.message);
}

getPackageManager

export function getPackageManager(pm?: string): PackageManager;

Gets the package manager to use, defaulting to "npm" if none specified.

Parameters

  • pm - Optional package manager name (defaults to "npm")

Returns

The package manager to use

Example

import { getPackageManager } from "npm-updater";

// Uses default npm
const defaultManager = getPackageManager(); // "npm"

// Uses specified package manager
const pnpmManager = getPackageManager("pnpm"); // "pnpm"

formatPackageManagerList

export function formatPackageManagerList(): string;

Returns a comma-separated string of all supported package managers.

Returns

Comma-separated list of supported package managers

Example

import { formatPackageManagerList } from "npm-updater";

const list = formatPackageManagerList();
console.log(list); // "npm, pnpm, yarn, bun"

üöÄ run Function

Main entry point function that processes command-line arguments and executes the appropriate command. 

export async function run(): Promise

Description

The run function processes command-line arguments from process.argv and executes the corresponding npm-updater command. It handles:

  • Package manager selection via --pm flag
  • Command parsing and routing
  • Help display for invalid or missing commands
  • Error handling and user feedback

Usage

import { run } from "npm-updater";

// Run with command-line arguments
await run();

// Or programmatically with custom arguments
process.argv = ["node", "npm-updater", "ls"];
await run();

Supported Commands

ls / list

List globally installed packages

update / upgrade / -u / --update

Update a specific package

updateall

Update all global packages

latestversion

Show latest version of a package

version / --version / -v

Show npm-updater version

help / --help / -h

Show help and usage information

about

Show information about npm-updater

üìù Command Functions

runls

export async function runls(packageManager?: string): Promise

Lists all globally installed packages using the specified package manager.

Parameters

  • packageManager - Optional package manager name (defaults to "npm") 
import { runls } from "npm-updater/commands/ls";

// List packages using npm (default)
await runls();

// List packages using pnpm
await runls("pnpm");

runupdate

export async function runupdate(
  inputPackageName?: string,
  packageManager?: string,
): Promise

Updates a specific package to its latest version using the specified package manager.

Parameters

  • inputPackageName - Name of the package to update (supports version specification)
  • packageManager - Optional package manager name (defaults to "npm") 
import { runupdate } from "npm-updater/commands/update";

// Update to latest version
await runupdate("prettier");

// Update to specific version
await runupdate("react@18.2.0");

// Update using pnpm
await runupdate("typescript", "pnpm");

runupdateall

export async function runupdateall(packageManager?: string): Promise

Updates all global packages to their latest versions using the specified package manager.

Parameters

  • packageManager - Optional package manager name (defaults to "npm") 
import { runupdateall } from "npm-updater/commands/updateall";

// Update all packages using npm (default)
await runupdateall();

// Update all packages using pnpm
await runupdateall("pnpm");

showlatestversion

export async function showlatestversion(
  packageName?: string,
  packageManager?: string,
): Promise

Displays the latest available version of a package using the specified package manager.

Parameters

  • packageName - Name of the package to check
  • packageManager - Optional package manager name (defaults to "npm") 
import { showlatestversion } from "npm-updater/commands/latestversion";

await showlatestversion("prettier");
await showlatestversion("react", "pnpm");

showversion

export function showversion(): void

Displays the current version of npm-updater. 

import { showversion } from "npm-updater/commands/version";

showversion(); // Displays: Version: 0.0.12

showabout

export async function showabout(): Promise

Displays information about npm-updater including name, version, description, and repository. 

import { showabout } from "npm-updater/commands/about";

await showabout();

üí° Programming Examples

Basic Usage

Simple programmatic usage to list and update packages. 

import { run } from "npm-updater";

// Use command line arguments
await run();

// Or call individual functions
import { runls, runupdate } from "npm-updater";

await runls("pnpm");
await runupdate("prettier", "npm");

Custom Configuration

Work with package manager configurations directly. 

import {
  getPackageManagerConfig,
  validatePackageManager,
} from "npm-updater";

const manager = validatePackageManager("pnpm");
const config = getPackageManagerConfig(manager);

console.log(config.installCommand("prettier"));
// Output: "pnpm add -g prettier@latest"

Error Handling

Handle errors gracefully in your applications. 

import { runupdate } from "npm-updater/commands/update";

try {
  await runupdate("non-existent-package");
} catch (error) {
  console.error("Update failed:", error.message);
}

// Validate package manager first
import { validatePackageManager } from "npm-updater";

try {
  const manager = validatePackageManager("unknown");
} catch (error) {
  console.error("Invalid package manager:", error.message);
}

Integration Example

Integrate npm-updater into your build scripts. 

import { runupdateall } from "npm-updater/commands/updateall";

async function updateDevDependencies() {
  console.log("Updating development tools...");
  await runupdateall("npm");
  console.log("Development tools updated!");
}

// In your build script
updateDevDependencies().catch(console.error);

Return Types

// Most commands return void as they output to console
type CommandResult = void;

// For programmatic use, you might want structured output
interface StructuredOutput {
  success: boolean;
  message?: string;
  error?: string;
  data?: unknown;
}

// Example usage pattern
async function executeCommandSafely(
  command: () => Promise
): Promise {
  try {
    await command();
    return { success: true };
  } catch (error) {
    return { 
      success: false, 
      error: error instanceof Error ? error.message : String(error) 
    };
  }
}
